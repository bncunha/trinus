# Agent Workflows

Fluxos operacionais para tarefas recorrentes.

Exemplos:

- Implementar uma feature.
- Revisar código.
- Criar endpoint.
- Criar tela Angular.
- Atualizar documentação.

## Workflows Disponíveis

- `define-feature`: refina uma feature antes da implementação.
- `implement-feature`: coordena implementação, testes, revisão, QA e documentação.
- `review-code`: orienta revisão de código.
- `run-qa`: orienta validação end-to-end.

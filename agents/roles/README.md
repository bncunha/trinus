# Agent Roles

Papéis dos agentes do projeto.

Cada arquivo deve descrever:

- Objetivo do agente.
- Responsabilidades.
- Limites de atuação.
- Entradas esperadas.
- Saídas esperadas.

## Agentes

- `orchestrator`: coordena a atuação dos demais agentes.
- `product-manager`: refina features, regras e critérios de aceite.
- `ux-designer`: define comportamento de telas e experiência de uso.
- `frontend-engineer`: implementa frontend Angular.
- `backend-engineer`: implementa backend Node.js com TypeScript.
- `qa-engineer`: valida fluxos com testes end-to-end.
- `code-reviewer`: revisa código, riscos, testes e contratos.
- `docs`: atualiza documentação do projeto.

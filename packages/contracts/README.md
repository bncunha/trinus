# Contracts

Contratos compartilhados entre frontend, backend e agentes.

Use este pacote para:

- DTOs.
- Tipos de request e response.
- Schemas de validação.
- Modelos de erro da API.
- Tipos de eventos entre partes do sistema.

Evite colocar aqui lógica de infraestrutura, acesso a banco, serviços Angular ou código específico da API.

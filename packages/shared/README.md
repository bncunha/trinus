# Shared

Utilitários e tipos genéricos compartilhados.

Use este pacote somente para código independente de aplicação, framework e infraestrutura.

Exemplos aceitáveis:

- Helpers de string, data e número.
- Tipos utilitários.
- Funções puras reutilizáveis.

Evite colocar aqui:

- Configuração de ambiente.
- Código de banco de dados.
- Componentes de UI.
- Serviços específicos do frontend ou backend.
- Regras de negócio com dependências de aplicação.

# Navegacao Inicial

## Gestor

Fluxo principal:

1. Acessa o sistema pelo Login.
2. Acessa o Dashboard operacional.
3. Identifica alertas, gargalos, criticidade e pedidos em risco.
4. Abre a Lista de Pedidos ou a Fila de Producao.
5. Analisa o detalhe de um pedido.
6. Ajusta etapa, criticidade ou posicao na fila quando necessario.
7. Acompanha impacto e risco.
8. Finaliza pedido.

## Administrador

Fluxo principal:

1. Cria uma nova conta.
2. O sistema cria a empresa e o primeiro usuario administrador.
3. Acessa o Dashboard operacional.
4. Configura tamanhos, unidades, variaveis, setores, etapas e templates de producao.
5. Cadastra novos usuarios para a propria empresa quando necessario.

## Operador

A visao operacional do Operador fica fora do primeiro MVP.

Quando entrar no escopo, o Operador deve acessar apenas informacoes necessarias para sua propria execucao.

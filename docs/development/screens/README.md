# Telas do MVP

Esta pasta descreve as telas iniciais do produto.

As telas devem seguir:

- Mobile first.
- Simplicidade.
- Coerencia entre fluxos.
- Instrucoes curtas quando houver duvida.
- Exposicao minima de dados conforme o papel do usuario.

## Telas

- `auth.md`: login e criacao de nova conta.
- `dashboard.md`: dashboard operacional unico.
- `orders.md`: lista, criacao e detalhe de pedidos.
- `production-queue.md`: fila de producao.
- `settings.md`: configuracoes operacionais.
- `users.md`: gestao de usuarios.
- `operator.md`: visao futura do operador.
- `post-sale.md`: pos-venda futuro.

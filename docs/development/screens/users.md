# Tela: Usuários

## Objetivo

Permitir que o Administrador cadastre usuários para acessar sua empresa.

## Usuários

- Administrador.

## Conteúdo

Lista de usuários:

- Nome.
- E-mail.
- Papel.
- Situação do acesso.

## Ações

- Criar usuário.
- Alterar papel.
- Ativar acesso.
- Desativar acesso.

## Criar Usuário

Campos:

- Nome.
- E-mail.
- Papel.
- Acesso ao sistema.

Papéis:

- Administrador.
- Gestor.
- Operador.

## Comportamento

- Usuários criados pertencem à mesma empresa do Administrador.
- O cadastro de acesso é opcional por usuário.
- Quando o usuário não tiver acesso, ele pode existir apenas como referência operacional.
- A tela deve explicar de forma curta o impacto de cada papel.

## Instrução Recomendada

```txt
O papel define quais informações o usuário pode ver e alterar.
```

# Tela: Operador

## Status

Fora do primeiro MVP.

## Objetivo Futuro

Permitir que o Operador visualize apenas o que precisa executar, com informacoes limitadas e sem dados comerciais sensiveis.

## Conteudo Futuro

O Operador podera visualizar:

- Numero do pedido.
- Cliente.
- Situacao atual.
- Data de inicio.
- Data de entrega.
- Produtos.
- Etapa relacionada a sua execucao.

## Restricoes Futuras

O Operador nao deve visualizar:

- Preco final da venda.
- Criticidade.
- Informacoes comerciais sensiveis.
- Dados que nao sejam necessarios para execucao.

## Observacao

Enquanto execucao por operador e ordens de servico estiverem fora do MVP, esta tela permanece apenas como direcao futura.
